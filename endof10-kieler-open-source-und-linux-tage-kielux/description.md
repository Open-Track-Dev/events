Das Ende von Windows 10 ist eine Chance für alle noch Zögernden, endlich Linux
auszuprobieren. Im Rahmen der Kielux bieten wir vom 18.-20. September für
Neugierige Vorträge, Beratung, Info-Flyer und Kielux-USB-Sticks mit dem
anfängerfreundlichen Linux Mint (zum Selbstkostenpreis + nur, solange der
Vorrat reicht) an. So könnt Ihr Euch informieren und Linux auf eurem Computer
sowohl unverbindlich ausprobieren als auch installieren.
