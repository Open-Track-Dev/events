Esta es la edición Córdoba Capital, Argentina del Flisol. El FLISoL (Festival
Latinoamericano de Instalación de Software Libre) es el evento de difusión de
software libre más grande de Latinoamérica. Se realiza desde 2005 y, desde
2008, tiene lugar el cuarto sábado de abril de cada año de manera simultánea
en numerosas ciudades de América Latina y España.
